# Pengelompokkan-Perlakuan-Kelapa-Sawit-Berdasarkan-Gen-Menggunakan-K-Means-Consensus-Clustering
This repository contains the source code for analyzing transcriptome data of 388 oil palm plants (sourced from NCBI). The project focuses on identifying gene expression patterns under various environmental conditions to support the development of adaptive and high-yielding oil palm varieties.
